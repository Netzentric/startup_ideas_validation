Key Insight: ตลาดที่ดีที่สุดมี 3 ลักษณะร่วม

1. Hyper-Local Moat

ตลาดที่ต้องการความรู้ท้องถิ่น (กฎหมาย, ภาษา, relationships) สร้าง barrier ธรรมชาติที่ Big Tech เข้าไม่ถึง

2. Network Effects

Waste (generator↔recycler), Construction (contractor↔sub), Ag (farmer↔buyer) — ยิ่งคนใช้มาก ยิ่งมี value

3. Regulatory Tailwind

รัฐบาลบังคับให้ทำ compliance, waste tracking, food traceability — ลูกค้าต้องจ่ายไม่ว่าจะอยากหรือไม่

Ideas ที่ไม่ผ่านเกณฑ์ (คะแนน < 8)

กลยุทธ์ Market Entry: Hybrid Approach

ตลาดไทย 70 ล้านคนเพียงพอสำหรับ test แต่ไม่พอสำหรับ venture-scale returns

Thailand-First เหมาะกับ:

Waste management, Construction PM, Compliance, Field service, QC manufacturing, Ag supply chain — ต้องการ localization + regulatory expertise

Global Launch ตั้งแต่วันแรก:

Fleet telematics, Embedded lending — technology-first plays ที่ไม่ต้องการ localization มาก
